A class which represents the chess board \& handles all updates to the board state. 

This class draws \& scales the board/pieces, handles click/keyboard events, checks move legality, and updates the locations of pieces

Attempts to move a target piece to a target square \& if move is completed successfully, adds it to the move history. 

Checks rules for check \& checkmate, and checks rules for special-\/case moves (i.\+e. castle moves) to determine if move is legal.~\newline
 If move is deemed illegal, it is not completed. If move is deemed legal, the following actions are taken based on move type\+:~\newline



\begin{DoxyItemize}
\item Normal moves\+:~\newline
 1) Target piece is moved to the given destination square ( \doxylink{classc_piece}{c\+Piece} position members are updated )~\newline
 2) \doxylink{structc_move}{c\+Move} instance for a normal move is created \& added to the move history~\newline

\item Capture moves\+:~\newline
 1) Enemy piece at destination square is removed from the board~\newline
 2) Target piece is moved to the given destination square ( \doxylink{classc_piece}{c\+Piece} position members are updated )~\newline
 3) \doxylink{structc_move}{c\+Move} instance for a capture move is created \& added to the move history~\newline

\item Castle moves\+:~\newline
 1) Target piece (king) is moved to the given destination square ( \doxylink{classc_piece}{c\+Piece} position members are updated )~\newline
 2) Secondary piece (rook) is moved to the appropriate destination square based on destination square of the king ( \doxylink{classc_piece}{c\+Piece} position members are updated )~\newline
 3) \doxylink{structc_move}{c\+Move} instance for a castle move is created \& added to the move history~\newline

\end{DoxyItemize}
